# PhDResearch — PhD research assistant — track papers, notes, citations, and experiments.

**Job 62** | **Package:** com.jnetai.phdresearch

## What It Does
PhD research assistant — track papers, notes, citations, and experiments.

## Features
- Track research papers, citations, and experiments
- Organise papers by status (To Read, Reading, Read)
- Rate papers and add notes
- Log experiment hypotheses and results
- About page with update checker

## Build
- Android API 26+, Kotlin, Room, KSP, ViewBinding + XML, Material3 Dark
